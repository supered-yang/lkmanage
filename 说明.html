链接mysql

  1. 使用python3 推荐安装mysqlclient

    pip install mysqlclient

    注意：在使用pycharm创建django应用后，此应用是创建在python虚拟环境中的，如果直接打开cmd运行安装文件，则会将mysqlclient模块安装在系统python环境中。如果用pycharm创建的django应用，需要在python虚拟环境中运行此命令，即在pycharm的Terminal中执行安装命令。

    python虚拟环境介绍：

    https://www.cnblogs.com/technologylife/p/6635631.html

  2. 修改数据库配置,settings.py文件

    DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'lkdb',   #数据库名
        'USER': 'root',    #用户名
        'PASSWORD': '',
        'HOST': 'localhost',
        'PORT': '3306',
        'CHARSET':'utf8',##设置字符集，不然会出现中文乱码
    }
}

引入bootstrap

1、在项目目录下创建static文件夹
2、将下载的bootstrap文件夹复制到static目录下
3、在项目settings.py文件内增加如下配置

    STATIC_URL = '/static/'

    STATICFILES_DIRS = (
        os.path.join(BASE_DIR, 'static'),
    )
4、引用本地bootstrap，示例：

<!DOCTYPE html>
{% load staticfiles %}
<html>
<head lang="en">
    <meta charset="UTF-8">
    <!-- 引入jQuery -->
    <script src="http://apps.bdimg.com/libs/jqueryui/1.10.4/jquery-ui.min.js"></script>
    <script src="http://code.jquery.com/jquery-latest.js"></script>
    <!-- 引入 Bootstrap -->
    <link href="{% static 'bootstrap/css/bootstrap-theme.css' %}" rel="stylesheet" type="text/css">
    <link href="{% static 'bootstrap/css/bootstrap.css' %}" rel="stylesheet" type="text/css">
    <script type="text/javascript" src="{% static 'bootstrap/js/bootstrap.js' %}"></script>
    <title>数据展示平台</title>
</head>
<body>
    <!-- bootstrap 特性容器 -->
    <div class="container">
        <h1>Hello, world! </h1>
    </div>
</body>
</html>
